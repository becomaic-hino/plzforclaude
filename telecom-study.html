MODULE 01

전자기파 기초

주파수, 파장, 전파 특성, 감쇠 — 통신의 물리적 토대

⚡ 전자기파란?

전기장과 자기장이 서로를 만들어내며 공간을 타고 퍼져나가는 에너지. 빛, 와이파이, 5G 신호 모두 같은 전자기파다 — 주파수가 다를 뿐.

💡 비유

돌을 물에 던지면 물결이 퍼져나간다. 전자기파도 같은 "파동"인데, 물이 필요 없다. 우주 진공에서도 퍼져나간다.

📐 핵심 공식

λ = c / f
λ(파장) = c(빛의 속도, 30만km/s) ÷ f(주파수) → 주파수↑ = 파장↓ = 반비례

이 하나의 공식이 통신 산업 전체를 지배한다:

대역주파수특성영업적 의미
Sub-1GHz700MHz 등멀리 감, 건물 투과농촌/전국망 커버리지용
Sub-6GHz3.5GHz 등속도↑ + 커버리지 균형5G 주력 대역, 대부분의 딜
mmWave28GHz초고속, 100m, 벽 못 뚫음삼성 핵심 강점 영역

MODULE 02

변조, 다중접속, 스펙트럼

데이터를 전파에 싣고, 여러 사용자가 나눠 쓰고, 주파수를 사고파는 방법

📡 변조 (Modulation) — QAM

💡 비유

전파 = 빈 트럭. 데이터 = 짐. 변조 = 짐을 트럭에 싣는 방법. QAM은 진폭+위상을 동시에 조절해 한 신호에 다량의 데이터를 싣는다.

변조비트/신호비유세대
QPSK2 bits신호등 4색3G/4G 기본
64QAM6 bits64색 크레용4G LTE
256QAM8 bits256색 모니터5G
1024QAM10 bits전문가 모니터5G-Advanced

핵심 트레이드오프: 변조 차수↑ = 더 빠르지만, 신호 품질(SNR)이 좋아야 함 → 빔포밍이 SNR을 올려줘야 고차 변조가 가능

👥 다중접속 — 세대별 진화

세대방식비유핵심
1GFDMA전용 창구채널 수 = 통화 수
2GTDMA/CDMA시간 나누기 / 다른 언어디지털화
3GWCDMA전원 동시 발화 + 코드퀄컴 특허 기반
4G/5GOFDMA뷔페 — 필요한 만큼 가져감자원 블록 배분 = 슬라이싱 기반

OFDMA의 핵심: 주파수를 수백 조각(자원 블록)으로 쪼개 사용자별로 필요한 만큼 배분. 이 자원 블록을 용도별로 묶으면 = 네트워크 슬라이싱.

🏗️ 스펙트럼 — 통신의 부동산

💡 비유

스펙트럼 = 땅. 통신사가 정부에서 경매로 사는 주파수 사용권. 미국 C-band 경매: $81B(약 100조원). 삼성이 파는 건 "장비"가 아니라 "비싸게 산 땅의 ROI를 극대화하는 방법"이다.

🎤 면접 킬러 멘트
"통신장비 B2B 영업의 핵심 가치 제안은 '스펙트럼 효율성'이라는 한 단어로 수렴합니다. 통신사가 경매에서 수조 원을 들여 산 주파수에서, 삼성의 Massive MIMO·빔포밍·256QAM·AI-RAN이 얼마나 더 많은 데이터를 뽑아내느냐가 곧 장비의 가치입니다."
MODULE 03

셀룰러 네트워크 구조

셀, 기지국 내부(RU/DU/CU), 핸드오버, 코어 네트워크

🐝 셀(Cell) — 벌집 구조

도시를 벌집처럼 작은 육각형으로 나누고 각각에 기지국 배치. 삼성 제품: 매크로 셀 → Massive MIMO Radio, 스몰 셀 → Link Cell(3.8L, 2.8kg 삼성 차별화 제품).

🍽️ 기지국 = RU + DU + CU

💡 레스토랑 비유

RU(홀 서버) = 전파 송수신, 손님과 직접 대면
DU(라인쿡) = 실시간 신호 처리, 즉시 조리
CU(총주방장) = 세션 관리·핸드오버, 전체 흐름 관리

전통 기지국: 3개가 한 박스 → 무겁고 비쌈. 5G 분리형: DU/CU를 소프트웨어로 → 이것이 vRAN의 기반.

🏃 핸드오버 — 이어달리기 바톤 터치

이동 중 한 셀→다음 셀로 끊김 없이 넘어가는 것. 삼성 Mobility Enhancer: mmWave 환경에서 핸드오버 성능 30% 향상.

🧠 코어 네트워크 — 도시의 중앙 관제센터

인증, 세션 관리, 라우팅, 과금, 슬라이싱을 담당. 4G 코어(EPC) = 일체형, 5G 코어(5GC) = 마이크로서비스 기반 클라우드 네이티브. 삼성 = vCore로 end-to-end SA 솔루션 제공.

🔗 전체 아키텍처

[단말] ←전파→ [RU] ──프론트홀── [DU] ──미드홀── [CU] ──백홀── [Core] ──→ 인터넷 ↑ ↑ ↑ ↑ 무선 구간 Fronthaul Midhaul Backhaul 삼성이 파는 것: RU + DU + CU (= RAN) + Core (= vCore) = end-to-end
MODULE 04

이동통신 세대별 진화 — 1G → 6G

기술 업그레이드가 아니라 비즈니스 모델의 전환

세대연도킬러앱비즈니스 변화
1G1984이동 전화"걸어다니며 통화"
2G1991문자(SMS)디지털화, 글로벌 로밍
3G2001모바일 인터넷브라우징, 앱스토어 초기
4G2009영상 스트리밍스마트폰 혁명
5G2019초저지연, IoTB2B 전환 — 공장, 자율주행
6G~2030디지털트윈, 홀로그램물리-디지털 융합

🎯 5G의 3대 시나리오 — 면접 필수

시나리오 1
eMBB 초고속

피크 20Gbps. 4K/8K, AR/VR. 소비자 대상, 4G의 연장선.

시나리오 2
URLLC 초저지연

1ms 이하, 99.999%. 자율주행, 원격수술. B2B의 핵심 가치.

시나리오 3
mMTC 대규모 IoT

1km²당 100만 기기. 스마트시티 센서, 물류 추적.

🔮 6G — 2030년

목표: 피크 1Tbps(50배), 지연 0.1ms(10배), THz 대역, AI 네이티브, NTN(위성+드론), 디지털 트윈, 센싱+통신 융합.
삼성 포지션: 2020년 업계 최초 6G 백서 발간. UCSB와 sub-THz 12Gbps 시연. "5G mmWave 양산 경험 → 6G THz로 자연 연장" 논리.

🎤 면접 킬러 멘트
"5G의 진짜 비즈니스 가치는 eMBB(초고속)가 아니라 URLLC(초저지연)입니다. '빠른 인터넷'은 ARPU를 높이지 못하지만, '1ms 전용 네트워크'는 프리미엄 과금이 가능합니다. 이 B2B 전환이 삼성 네트워크사업부가 해외영업을 강화하는 이유입니다."
MODULE 05

5G 핵심 기술 심화

Massive MIMO, 빔포밍, mmWave, 네트워크 슬라이싱을 하나의 시스템으로

📊 Massive MIMO의 3가지 이득

어레이 게인

64개 안테나 동시 송출 → 신호 세기 64배(≈18dB) 증가. 같은 출력으로 훨씬 먼 거리.

공간 다중화

16명에게 동시에 다른 데이터 전송(MU-MIMO). 셀 용량 16배 효과.

간섭 제거

원치 않는 방향 신호를 "null 빔"으로 억제. 인접 셀 간섭 감소.

🎯 3D 자원 관리

공간(빔포밍) × 주파수(OFDMA) × 시간(스케줄링)
Massive MIMO 기지국이 매 밀리초마다 수행하는 3차원 자원 최적화. AI-RAN이 여기에 투입.

🔪 네트워크 슬라이싱 (Q24 직접 대응)

💡 고속도로 전용차로 비유

1차로 = URLLC 슬라이스 (자율주행, 원격수술) — 초저지연
2~3차로 = eMBB 슬라이스 (스마트폰, 영상) — 고속
갓길 = mMTC 슬라이스 (IoT 센서) — 저전력, 소량

비즈니스 임팩트: 각 슬라이스에 다른 SLA, 다른 가격 → 통신사의 B2B 프리미엄 매출원. 삼성 = vRAN + vCore로 end-to-end SA 슬라이싱 지원.

MODULE 07

vRAN / Open RAN / AI-RAN

삼성 네트워크사업부의 3대 차별화 무기

🎮 vRAN — 게임 콘솔에서 게이밍 PC로

전통 RAN = PS5 (전용 하드웨어에서만 동작).
vRAN = 게이밍 PC (범용 서버에서 소프트웨어로 동작). 업그레이드 = SW 업데이트.

MODULE 08 · 최종

기술 → 비즈니스 연결

TCO, 수주 구조, 고객 가치 제안 — 기술을 "통신사 CFO가 듣고 싶은 언어"로

💰 TCO (Total Cost of Ownership)

CapEx (30~40%)OpEx (60~70%)
정의장비 구매, 인프라 건설유지보수, 전기, 인건비, 임대
비유집 사는 돈관리비+전기+수리비
주기5~10년마다매달 발생

핵심: 장비 가격(CapEx)만 비교하면 화웨이가 가장 싸다. 하지만 10년 TCO로 보면 OpEx를 줄이는 삼성 기술(vRAN, AI-RAN, 소형화)이 경쟁력.

📋 수주 프로세스

1. RFI (정보 요청) → 삼성: 기술 역량·레퍼런스 소개 2. RFP/RFQ (제안 요청) → 삼성: 기술+가격 제안서 + TCO 분석 3. PoC (필드 트라이얼) → 실제 환경에서 삼성 장비 테스트 (수 개월) 4. 벤더 선정 + 계약 → Verizon 딜: $6.65B 다년 계약 5. 배포 (Deployment) → 단계적 롤아웃 (수 개월~수 년) 6. 유지보수 + 확장 → 장기 지원, SW 업데이트, 추가 수주

영업 담당자의 역할: 이 전체 과정을 리드하며, 기술팀-고객 간 "기술의 비즈니스 번역가" 역할.

🎤 최종 통합 면접 답변 — "삼성 네트워크의 경쟁력은?"
"세 가지입니다. 첫째, 기술적 연속성 — 2009년 mmWave 빔포밍 연구부터 2020년 업계 최초 6G 백서까지, 매 단계마다 선도적 위치를 확보했습니다. 둘째, 'Open이면서 Complete' — O-RAN 호환 멀티벤더와 풀스택 단일 벤더를 동시에 제공하는 유일한 벤더입니다. 셋째, 지정학적 포지션 — 화웨이가 서방권에서 배제되는 흐름 속에, 유일한 비유럽·비중국 풀스택 통신장비 벤더입니다."
🎤 "비전공자로서 기술 이해가 부족하지 않을까요?"
"통신장비 B2B 영업에서 가장 중요한 건 기술 자체를 개발하는 능력이 아니라, 기술의 가치를 고객의 비즈니스 언어로 번역하는 역량입니다. 빔포밍을 수식으로 설명하는 건 엔지니어의 일이지만, 빔포밍이 통신사의 스펙트럼 ROI를 어떻게 높이는지 설명하는 건 영업의 일입니다."
